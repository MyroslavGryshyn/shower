			import sys.getsizeof
			from sys import getsizeof
			import sys as sys2

Try/except/finally

  • finally виконується при виході з блоку try, не залежно від вийнятків
  • finally виконується при наявності break або continue
  • finally виконується при вийнятку в exception
  • finally виконується до return в блоці try чи except

Менеджер контексту

При роботі з ресурсом(файл, сокет і т.д.) його потрібно звільнити 

      with open("test.txt", "w") as f:
          f.write("the Answer is: ")

Вкладені менеджери контексту

      with open("test.txt", "w") as t, \
           open("index.html", 'r') as i:
          i.read()
